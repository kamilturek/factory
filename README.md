# Multi-threaded Factory
![CI](https://github.com/kamilturek/factory/workflows/CI/badge.svg)

## build
```
$ mkdir build
$ cd build
$ cmake ..
$ make
```

## CMake Options

#### CMAKE_BUILD_TYPE
- Default: Debug

#### ENABLE_SANITIZER_ADDRESS
- Default: OFF

#### ENABLE_SANITIZER_MEMORY
- Default: OFF

#### ENABLE_SANITIZER_THREAD
- Default: OFF

#### ENABLE_SANITIZER_UNDEFNED_BEH
- Default: OFF
